# 06: アカウント一覧表示・検索（F3）

## 概要

登録済みアカウント情報をカードリストで一覧表示し、サービス名での検索機能を提供する。

## 対応する要件

- REQUIREMENTS.md ⑤ F3: アカウント一覧表示
- REQUIREMENTS.md ⑥ ホーム（一覧）画面

## タスク

- [ ] ホーム画面（`app/(auth)/index.tsx`）の作成
- [ ] アカウントカードのリスト表示（FlatList、`keyExtractor` / `getItemLayout` 設定）
- [ ] 登録日時の降順ソート（最新が上）
- [ ] `SearchBar` コンポーネントの作成
- [ ] サービス名でのインクリメンタルサーチ実装
- [ ] 新規追加 FAB（Floating Action Button）の配置
- [ ] 空状態の表示（アカウント未登録時のガイダンス）
- [ ] ローディング表示（スケルトン UI またはスピナー）
- [ ] Pull-to-Refresh による再読み込み
- [ ] カードタップで詳細画面へ遷移

## 完了条件

- 登録済みアカウントがカード形式で一覧表示される
- 最新の登録が上に表示される
- サービス名で絞り込み検索ができる
- FAB タップで新規登録画面へ遷移する
- リストが大量でもスムーズにスクロールできる
