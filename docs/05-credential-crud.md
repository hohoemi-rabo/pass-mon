# 05: アカウント情報の CRUD（F2）

## 概要

アカウント情報（サービス名、ID、パスワード、PIN、メモ）の登録・表示・編集・削除機能を実装する。

## 対応する要件

- REQUIREMENTS.md ⑤ F2: アカウント情報の登録・編集・削除
- REQUIREMENTS.md ⑥ 新規登録画面 / 詳細・編集画面

## タスク

- [ ] CRUD 操作フック `useCredentials` の作成（Supabase RPC 経由で暗号化・復号）
- [ ] 新規登録画面（`app/(auth)/add.tsx`）の作成
  - [ ] サービス名（必須）、アカウント ID、パスワード、PIN コード、メモのフォーム
  - [ ] バリデーション（サービス名の必須チェック）
  - [ ] 保存時に暗号化関数を経由して DB に保存
- [ ] 詳細・編集画面（`app/(auth)/[id].tsx`）の作成
  - [ ] パスワード / PIN の表示・非表示切り替え（目のアイコン）
  - [ ] ワンタップコピー機能（`expo-clipboard`）
  - [ ] 編集モードへの切り替え
  - [ ] 削除ボタン + 確認ダイアログ
- [ ] `CredentialCard` コンポーネントの作成（一覧で使用するカード UI）
- [ ] `PasswordField` コンポーネントの作成（マスク表示 / 表示切替 / コピー）
- [ ] 入力フォームのシニア向け最適化（大きなフォント、明確なラベル、十分なタップ領域）

## 完了条件

- アカウント情報を新規登録できる
- 登録した情報を詳細画面で閲覧・編集・削除できる
- パスワード / PIN がデフォルトで非表示、タップで表示切替できる
- ワンタップでクリップボードにコピーできる
- 削除時に確認ダイアログが表示される
- DB 上のパスワード / PIN が暗号化されていることを確認
