# 05: アカウント情報の CRUD（F2） [DONE]

## 概要

アカウント情報（サービス名、ID、パスワード、PIN、メモ）の登録・表示・編集・削除機能を実装する。

## 対応する要件

- REQUIREMENTS.md ⑤ F2: アカウント情報の登録・編集・削除
- REQUIREMENTS.md ⑥ 新規登録画面 / 詳細・編集画面

## タスク

- [x] CRUD 操作フック `useCredentials` の作成（Supabase RPC 経由で暗号化・復号）
- [x] 新規登録画面（`app/(auth)/add.tsx`）の作成
  - [x] サービス名（必須）、アカウント ID、パスワード、PIN コード、メモのフォーム
  - [x] バリデーション（サービス名の必須チェック）
  - [x] 保存時に暗号化関数を経由して DB に保存
- [x] 詳細・編集画面（`app/(auth)/[id].tsx`）の作成
  - [x] パスワード / PIN の表示・非表示切り替え（目のアイコン）
  - [x] ワンタップコピー機能（`expo-clipboard`）
  - [x] 編集モードへの切り替え
  - [x] 削除ボタン + 確認ダイアログ
- [x] `CredentialCard` コンポーネントの作成（一覧で使用するカード UI）
- [x] `PasswordField` コンポーネントの作成（マスク表示 / 表示切替 / コピー）

## 完了条件

- アカウント情報を新規登録できる
- 登録した情報を詳細画面で閲覧・編集・削除できる
- パスワード / PIN がデフォルトで非表示、タップで表示切替できる
- ワンタップでクリップボードにコピーできる
- 削除時に確認ダイアログが表示される
- DB 上のパスワード / PIN が暗号化されていることを確認
