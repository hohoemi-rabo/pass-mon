# パスもん — 初回セットアップ手順書

> **対象:** ほほえみラボの生徒さん（Android）/ 家族の方
> **最終更新:** 2026年4月12日

---

## 1. アプリのインストール（Android）

### 1-1. ダウンロード

1. まさゆきさんから共有された **ダウンロードリンク** をタップします
2. 「ダウンロード」をタップします
3. ダウンロードが完了したら通知バーを引き下げ、ダウンロードしたファイルをタップします

### 1-2. インストールの許可

初回のみ「提供元不明のアプリ」の許可が必要です:

1. 「このソースからのアプリのインストールを許可しますか？」と表示されたら **「許可」** をタップ
2. 「インストールしますか？」と表示されたら **「インストール」** をタップ
3. インストールが完了したら **「開く」** をタップ

> **ポイント:** Play ストアからのインストールではないため、この許可が必要です。安全なアプリですのでご安心ください。

---

## 2. ログイン

1. アプリを開くと **「パスもん」** のロゴが表示されます
2. **「Googleでログイン」** ボタンをタップします
3. お使いの **Google アカウント**（Gmail のアドレス）を選択します
4. ログインが完了すると、ホーム画面が表示されます

> **ポイント:** Google アカウントのパスワードを忘れた場合は、教室で一緒に確認しましょう。

---

## 3. アカウント情報の登録

### 3-1. 新規登録

1. ホーム画面右下の **オレンジの＋ボタン** をタップします
2. 以下の情報を入力します:

| 項目 | 入力例 | 必須？ |
|------|--------|--------|
| サービス名 | Gmail、Amazon、楽天 | **必須** |
| アカウントID | メールアドレスやユーザー名 | 任意 |
| パスワード | そのサービスのパスワード | 任意 |
| PINコード | PCのログインPINなど | 任意 |
| メモ | 自由にメモ | 任意 |

3. 入力が終わったら **「保存する」** をタップします

### 3-2. 登録した情報の確認

1. ホーム画面のカードをタップすると、詳細画面が開きます
2. パスワード・PINは **●●●●●●●●** で隠れています
3. **目のアイコン** をタップすると表示されます
4. **コピーアイコン** をタップすると、クリップボードにコピーされます

### 3-3. 編集・削除

- 詳細画面で **「編集する」** をタップすると、内容を変更できます
- **「削除する」** をタップすると、確認の後に削除されます

---

## 4. 家族共有の設定

### 4-1. 家族を招待する（生徒さん側）

1. 下のタブから **「共有」** をタップします
2. **「家族を招待する」** ボタンをタップします
3. 6文字の **招待コード** が表示されます
4. **「LINEやメールで共有」** をタップして、家族に招待コードを送ります

### 4-2. 招待を受ける（家族側）

1. パスもんアプリをインストールします（手順1と同じ）
2. Google アカウントでログインします
3. 下のタブから **「共有」** をタップします
4. **「招待コード」** の欄に、受け取ったコードを入力します
5. **「招待を受ける」** をタップします
6. 完了すると、生徒さんのアカウント情報が閲覧できるようになります

> **ポイント:** 家族の方は **閲覧のみ** です。編集・削除はできません。

### 4-3. 共有を解除する（生徒さん側）

1. 共有タブで **「共有を解除する」** をタップします
2. 確認ダイアログで **「解除する」** をタップします

---

## 5. アプリの更新

新しいバージョンが出た場合:

1. まさゆきさんから **新しいダウンロードリンク** が共有されます
2. リンクをタップして APK をダウンロードします
3. **「インストール」** をタップすると、既存のアプリが上書き更新されます
4. データはそのまま保持されます（消えません）

---

## 6. よくあるご質問

### Q: パスワードを忘れてログインできない
→ Google アカウントのパスワードのことです。教室で一緒に復旧しましょう。

### Q: アプリを消してしまった
→ もう一度インストールすれば、Google アカウントでログインするとデータが戻ります。

### Q: 機種変更した
→ 新しいスマホにアプリをインストールし、同じ Google アカウントでログインすればデータが引き継がれます。

### Q: 家族が見られなくなった
→ 招待コードの期限が切れた可能性があります。共有タブから再度招待してください。

## iOS（iPhone）をお使いの方へ

iPhone の方は TestFlight を通じて配布します:

1. まさゆきさんから **TestFlight の招待メール** が届きます
2. App Store から **TestFlight** アプリをインストールします
3. 招待メールの **「テストを開始」** をタップします
4. TestFlight アプリからパスもんをインストールします

以降の使い方は Android と同じです。

## 開発者向けメモ

### APK ビルド手順

```bash
# EAS にログイン
eas login

# 環境変数を EAS Secrets に登録（初回のみ）
eas secret:create --name EXPO_PUBLIC_SUPABASE_URL --value "<url>"
eas secret:create --name EXPO_PUBLIC_SUPABASE_ANON_KEY --value "<key>"

# Android APK ビルド
eas build --platform android --profile preview

# iOS ビルド（TestFlight 用）
eas build --platform ios --profile production
eas submit --platform ios
```

### 配布更新フロー

1. コードを修正・コミット
2. `eas build --platform android --profile preview` で新しい APK をビルド
3. EAS ダッシュボードから新しいダウンロード URL を取得
4. 生徒さんに新しいリンクを共有
