# 08: ナビゲーション・画面レイアウト構成

## 概要

Expo Router を使ったファイルベースルーティングの最終構成を整備する。タブナビゲーション、認証ガード、モーダル表示を含む。

## 対応する要件

- REQUIREMENTS.md ⑥ 画面構成
- REQUIREMENTS.md ⑬ プロジェクト構成

## タスク

- [ ] ルーティング構成の整理
  - [ ] `app/_layout.tsx` — ルートレイアウト（`Stack.Protected` で認証ガード）
  - [ ] `app/login.tsx` — ログイン画面
  - [ ] `app/(auth)/_layout.tsx` — 認証済みタブナビゲーション
  - [ ] `app/(auth)/index.tsx` — ホーム（一覧）
  - [ ] `app/(auth)/add.tsx` — 新規登録
  - [ ] `app/(auth)/[id].tsx` — 詳細・編集
  - [ ] `app/(auth)/share.tsx` — 家族共有設定
  - [ ] `app/(auth)/settings.tsx` — 設定
- [ ] タブナビゲーションのアイコン・ラベル設定
- [ ] 設定画面の作成（ログアウト、バージョン情報）
- [ ] テンプレートの `(tabs)` グループから `(auth)` グループへの移行
- [ ] ヘッダーのスタイリング（カラーパレット適用）

## 完了条件

- 全画面間のナビゲーションが正常に動作する
- 未ログイン時はログイン画面のみアクセス可能
- タブナビゲーションで主要画面を切り替えられる
- 設定画面からログアウトできる
