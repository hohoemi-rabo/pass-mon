# 07: 家族共有機能（F4）

## 概要

「もしもの時」に家族がアカウント情報を閲覧できる共有機能を実装する。

## 対応する要件

- REQUIREMENTS.md ⑤ F4: 家族共有機能
- REQUIREMENTS.md ⑥ 家族共有設定画面

## タスク

- [ ] 家族共有設定画面（`app/(auth)/share.tsx`）の作成
- [ ] 招待コード生成ロジックの実装（ユニークなコード発行）
- [ ] 招待リンクの生成（ディープリンク `passmon://invite/{code}`）
- [ ] LINE / メールでの招待リンク共有（`expo-sharing` または Share API）
- [ ] 招待リンクからのアプリ起動・招待受諾フロー
- [ ] 共有状態の表示（未招待 / 招待中 / 共有中）
- [ ] 共有解除機能（解除ボタン + 確認ダイアログ）
- [ ] 家族側の閲覧専用 UI（編集・削除ボタンを非表示）
- [ ] 共有先の最大人数制限（MVP: 1名）
- [ ] `family_shares` の status 管理（pending → active → revoked）

## 完了条件

- 招待リンクを生成して LINE / メールで共有できる
- 家族がリンクからアプリにアクセスし、閲覧のみ可能になる
- 家族からは編集・削除ができない
- 生徒さん側からいつでも共有を解除できる
- 共有状態が画面上で確認できる
