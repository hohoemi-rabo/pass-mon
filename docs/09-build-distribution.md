# 09: ビルド・配布・テスト

## 概要

EAS Build で APK を生成し、生徒さんへの配布フローを確立する。教室での初回セットアップ支援を含む。

## 対応する要件

- REQUIREMENTS.md ⑩ 配布・運用計画

## タスク

- [ ] `eas build --platform android --profile preview` で APK を生成
- [ ] APK のダウンロード URL を取得し、QR コードを生成
- [ ] Android 端末での動作確認（インストール、ログイン、CRUD、共有）
- [ ] iOS ビルド（TestFlight 経由、2名向け）の設定と配布
- [x] 教室での初回セットアップ手順書の作成（`docs/setup-guide.md`）
  - [x] APK インストール手順
  - [x] Google ログイン手順
  - [x] 初回アカウント登録の手順
  - [x] 家族共有の設定手順
- [x] アプリ更新時の配布フロー確認（新 APK のリンク共有）
- [ ] 端末間の同期動作確認（複数端末で同一アカウント）

## 完了条件

- APK が正常に生成され、ダウンロードリンクで配布できる
- Android 端末で全機能が正常に動作する
- iOS で TestFlight 経由の配布（またはExpo Go）が動作する
- 初回セットアップ手順書が完成している
