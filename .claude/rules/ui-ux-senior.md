# UI/UX ルール（シニア対応）

## デザインコンセプト

「ポップで親しみやすく、見やすく、迷わない」

## サイズ・アクセシビリティ

- フォントサイズ最小18px、見出し22px以上
- タップ領域最小48x48dp
- WCAG AA準拠のコントラスト比
- カード型UI、角丸、やさしい影

## カラーパレット

| 用途 | カラー |
|------|--------|
| プライマリ | `#FF8C42`（温かいオレンジ） |
| セカンダリ | `#4ECDC4`（ポップなミント） |
| 背景 | `#FFF8F0`（やわらかいクリーム） |
| カード背景 | `#FFFFFF` |
| テキスト | `#333333` |
| サブテキスト | `#888888` |
| 危険（削除等） | `#FF6B6B` |
| 成功 | `#51CF66` |
